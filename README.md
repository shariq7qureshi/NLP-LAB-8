# ✍️ AI Grammar & Writing Assistant

A localized web application built using **Streamlit** and **Hugging Face Transformers** that acts as an editing assistant. The system processes text modifications directly on the server host without depending on external proprietary LLM APIs.

## 🚀 Key Features
* **Fix Grammar:** Instantly corrects syntactic, spelling, and structural mistakes.
* **Professional Rewrite:** Upgrades text context to match a corporate, formal tone.
* **Simplify Text:** Demystifies complex corporate terminology into clear language.
* **Summarize:** condenses large paragraphs down to core talking points.
* **Convert to Email:** Transitions raw thoughts or notes into structured messages.

## 🛠️ Tech Stack & Constraints
* **Frontend UI:** Streamlit (v1.45.1)
* **ML Backend Framework:** TensorFlow (CPU Only Optimization)
* **Model Checkpoint:** `google/flan-t5-base`
* **Python Target:** 3.11
